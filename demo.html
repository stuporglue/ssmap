This is a stupid simple map (SSMap) demo!

Leaflet maps are already pretty easy to use, but what if you want it EVEN EASIER?

The stupid simple map puts a leaflet map on your web page, automatically loading Leaflet if needed.

It defaults to a 500px square map of the world, centered at (0,0). You can change:

  • The zoom
  • The center
  • Map dimensions
  • Which basemap to use (street or satelite view)
  • If a marker is placed at the centerpoint of the map

Simply put this line in your HTML where you want a map to appear:

<script src='http://stuporglue.github.io/ssmap/ssmap-min.js'></script>

You can also specify several options

<script src='http://stuporglue.github.io/ssmap/ssmap-min.js?lat=-90&lon=45&z=8&mark=true&tiles=map&w=50%'></script>

You can set up a div for the map if you want and set extra styles or whatever. 

<script src='http://stuporglue.github.io/ssmap/ssmap-min.js?div=floatydiv&lat=-90&lon=45'></script>

Options

All options are optional.

OptionDefaultValuesNotes
lat0Latitude (decimal degrees)-90 to 90
lon0Longitude (decimal degrees)-180 to 180
divrandom IDstringMap div ID. The div also get class ssmap
h500pxValid CSS heightHeight of the map div (including units)
w500pxValid CSS WidthWidth of the map div (including units)
z11-18Initial zoom level
tilessatmap (street map) or sat (satelite)Which tiles to use
marktt (true) or f (false)Place a marker at the map centerpoint if coordinates are not set to 0,0?

Advanced Usage

There is no advanced usage. Leaflet is really easy to use, so switch to that. Check out the examples and API.
